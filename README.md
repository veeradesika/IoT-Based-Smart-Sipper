# IoT-Based Smart Sipper

## 📌 Project Overview
The IoT-Based Smart Sipper is an intelligent water bottle designed to monitor water temperature and provide automated heating and cooling using IoT technology. The system uses an ESP32 microcontroller, temperature sensors, and a Peltier module to maintain the desired water temperature while enabling real-time monitoring through the cloud.

## 🎯 Objectives
- Monitor water temperature in real time.
- Automatically heat or cool water based on user settings.
- Enable IoT-based remote monitoring.
- Promote convenient and smart hydration.

## 📂 System Workflow
1. The temperature sensor continuously measures the water temperature.
2. ESP32 processes the sensor data.
3. Based on predefined temperature limits, the Peltier module switches between heating and cooling.
4. Temperature data is displayed on the LCD.
5. Sensor readings are uploaded to Adafruit IO for real-time monitoring.
