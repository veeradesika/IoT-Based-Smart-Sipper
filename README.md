# IoT-Based Smart Sipper

## Project Description
The IoT-Based Smart Sipper is a smart water bottle that monitors water temperature and hydration levels using IoT technology. It can heat or cool water using a Peltier module and displays real-time data.

## Features
 Temperature monitoring
 Heating and cooling system
 Water level monitoring
 IoT-based real-time monitoring
 ESP32 microcontroller

## Components Used
 ESP32
 Peltier Module
 Temperature Sensor
 Water Level Sensor
 LCD Display
 Power Supply
